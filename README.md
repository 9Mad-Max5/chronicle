# Chronicle Audiobook Player

### Features

 - Sync audiobook progress on device
 - Support for file formats: mp3, m4a, m4b
 - Adjustable playback speed
 - Sleep timer
 - Skip silent audio
 - Offline support
 - Basic Android Auto support (playback works but no voice support)
